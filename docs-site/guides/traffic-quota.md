# 流量额度

控制面按 Agent 节点统计流量，支持设置月度额度和超额阻断。

## 查看流量

面板首页的仪表盘展示流量概览：已用 / 总额度、已阻断节点数、流量趋势图。

![仪表盘首页](/screenshots/panel-dashboard.png)

## 配置流量策略

在节点详情页设置：

| 字段 | 说明 |
| --- | --- |
| 计数方向 | 哪些流量计入额度：入站（rx）、出站（tx）、双向、取大值、混合公式 |
| 计费起始日 | 月度周期重置日期（1–31），默认 1 号 |
| 月度额度 | 每月最大流量。留空表示不限制 |
| 超额阻断 | 开启后达到额度会阻断该节点 |

节点被阻断后，面板首页和节点列表会显示阻断状态。下个计费周期自动恢复。你也可以手动调高额度或关闭超额阻断让节点立即恢复。

## 手动校准

以下情况需要手动校准流量统计：
- 更换网卡
- 重装操作系统
- 迁移数据到新服务器

在节点详情页操作：设置基准偏移（增减固定量），或清除当前周期（归零）。校准后所有统计使用校准值。

## 关闭流量统计

```ini
NRE_TRAFFIC_STATS_ENABLED=false
```

深入了解流量统计的实现原理，见 [流量统计原理](../reference/traffic-accounting.md)。
