Nginx-Reverse-Emby用自己的 VPS 反代 Emby 源站

面向已经有优化线路 VPS、购买或加入公费服/公益服 Emby/Jellyfin 的用户，按教程部署后把观看入口固定到自己的域名。

添加 HTTP 规则
GitHub

先部署到 VPS

准备能访问源站的优化线路 VPS，改好 API_TOKEN，用 Docker Compose 启动控制面和 local 节点。

再添加 HTTP 规则

在真实面板里选择 local，填写自己的加速入口域名和公费服/公益服给出的 Emby 源站地址。

最后换入口观看

配好 DNS 和端口后，播放器访问你的域名，由 VPS 连接源站，减少观看时必须挂代理的麻烦。
